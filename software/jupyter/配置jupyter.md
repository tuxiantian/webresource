# Jupyter Notebook 工作环境配置

背景

为什么要配置jupyter的工作环境呢，因为可以更方便的找到自己写的脚本所在的文件

步骤

1、新建一个文件夹，随便在哪里建，例如我在D盘新建了一个文件夹：D:\pythonworkspace

2、接下来配置环境变量，新建一个变量WORKPATH，变量值为：D:\pythonworkspace
